# Mobile Shop — Consultancy Project

This workspace contains a simple billing & stock management prototype.

Folders:
- backend — Express API (port 3000)
- frontend — React (Vite)

Quick start (PowerShell):

```powershell
# Backend
cd backend
npm install
npm start

# In a second terminal, frontend
cd frontend
npm install
npm run dev
```

Open the frontend dev server URL shown by Vite (usually http://localhost:5173) and ensure the backend runs on http://localhost:3000.
